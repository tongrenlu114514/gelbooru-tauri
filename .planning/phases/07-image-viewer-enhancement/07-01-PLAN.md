Create src/components/viewer/ImageViewer.vue with the following implementation:

**Imports:**
- Vue composition API: ref, computed, onMounted, onBeforeUnmount, watch
- naive-ui: NButton, NIcon, NText
- icons: ChevronBackOutline, ChevronForwardOutline, TrashOutline, ExpandOutline from @vicons/ionicons5
- convertFileSrc from @tauri-apps/api/core

**Props:**
```typescript
interface Props {
  images: ImageInfo[];
  visible: boolean;
  initialIndex: number;
}
```

**Emits:**
```typescript
emit('update:visible', false);
emit('delete', props.currentIndex);
```

**State:**
- currentIndex = ref(props.initialIndex || 0)
- zoomLevel = ref(1) // 1 = 100%
- panOffset = ref({ x: 0, y: 0 }) // in pixels
- isDragging = ref(false)
- dragStart = ref({ x: 0, y: 0 })

**Zoom behavior (per D-01):**
- Wheel zoom: wheel event handler, deltaY > 0 = zoom out (-0.1), else zoom in (+0.1)
- Zoom range: Math.max(0.5, Math.min(5, zoomLevel)) // 50% to 500%
- Keyboard zoom: '+' or '=' = zoomIn(), '-' = zoomOut(), '0' = resetZoom()

**Pan behavior (per D-02):**
- Only active when zoomLevel > 1
- mousedown starts drag (if zoomLevel > 1)
- mousemove updates panOffset: { x: e.clientX - dragStart.x, y: e.clientY - dragStart.y }
- mouseup ends drag
- Cursor: 'grab' when zoomed, 'grabbing' while dragging, 'default' when not zoomed

**Keyboard handler (per D-04):**
- ArrowLeft = prevImage()
- ArrowRight = nextImage()
- Escape = emit('update:visible', false)
- '+'/'=' = zoomIn()
- '-' = zoomOut()
- '0' = resetZoom()

**Image transform:**
```typescript
const imageStyle = computed(() => ({
  transform: `scale(${zoomLevel.value}) translate(${panOffset.value.x}px, ${panOffset.value.y}px)`,
  transformOrigin: 'center center',
}));
```

**Reset zoom on image change:**
When currentIndex changes, reset zoomLevel to 1 and panOffset to { x: 0, y: 0 }.

**Template structure:**
- Fullscreen overlay div (position: fixed, inset: 0, background: rgba(0,0,0,0.95))
- Click overlay to close viewer
- Centered image with transform style
- Navigation buttons: prev (left), next (right)
- Counter text: "{{ currentIndex + 1 }} / {{ images.length }}"
- Delete button (trash icon)
- Zoom controls display: show current zoom level percentage

**CSS requirements:**
- Overlay: z-index 9999, cursor pointer on background
- Image container: overflow hidden, cursor grab/grabbing/default
- Navigation buttons: quaternary circle style, fixed position left/right
- Filmstrip NOT included in this component (deferred to Task 3)

**Auto-pan activation:**
When zoomLevel > 1, automatically enable grab cursor. No need to hold mouse button - click and drag works.

**No new dependencies** - use pure CSS transforms and native browser events.

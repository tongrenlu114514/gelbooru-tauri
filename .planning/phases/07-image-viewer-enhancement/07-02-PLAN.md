**State:**

- Half range: Math.floor(9 / 2) = 4

**Computed visible thumbnails:**
```typescript
const visibleThumbnails = computed(() => {
  const half = Math.floor(9 / 2); // 4
  const start = Math.max(0, props.currentIndex - half);
  const end = Math.min(props.images.length - 1, props.currentIndex + half);
  // Return array with { index, image } for each visible thumbnail
  return Array.from({ length: end - start + 1 }, (_, i) => ({
    index: start + i,
    image: props.images[start + i],
  }));
});
```

**Thumbnail src:**
```typescript
function thumbSrc(path: string): string {
  return convertFileSrc(path.replace(/\\/g, '/'));
}
```

**CSS:**
- Container: fixed at bottom of viewport, height ~80px, horizontal scroll if needed
- Thumbnails: ~60px width, auto height (max 80px), object-fit cover
- Current thumbnail: highlight border or opacity change
- Hover: scale or border highlight
- Gap: 8px between thumbnails
- Background: dark semi-transparent (rgba(0,0,0,0.8))

**Template:**
```html
<div class="filmstrip" @click.self>
  <div class="filmstrip-track">
    <div
      v-for="item in visibleThumbnails"
      :key="item.index"
      class="filmstrip-thumb"
      :class="{ active: item.index === currentIndex }"
      @click="emit('select', item.index)"
    >
      <img :src="thumbSrc(item.image.path)" :alt="item.image.name" />
    </div>
  </div>
</div>
```

**Auto-scroll to center:**
When currentIndex changes, scroll the track so the active thumbnail is centered. Use a ref to the track container and scrollLeft calculation.

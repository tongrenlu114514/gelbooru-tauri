## Component Inventory

### TagAutocompleteInput (wrapper around NAutoComplete)

| State | Behavior |
|-------|----------|
| Default | Renders n-input with n-auto-complete overlay |
| Typing | Filter suggestions from search history + Gelbooru API |
| Dropdown open | Shows up to 8 suggestions, keyboard navigable |
| Item selected | Add tag to selectedTags, clear input |
| No matches | Show "无匹配标签" in dropdown |
| Loading | Show n-spin in dropdown |

**Interaction patterns:**
- Enter: select highlighted suggestion
- ArrowUp/Down: navigate suggestions
- Escape: close dropdown without selection
- Click on suggestion: select and add tag
- Min input 2 chars before triggering autocomplete

### SearchHistoryStore (Pinia store for tag frequency)

| Operation | Behavior |
|-----------|----------|
| recordSearch(tag) | Increment frequency for tag, persist to localStorage |
| getTopTags(limit) | Return top N tags by frequency |
| clearHistory() | Reset all frequencies |
